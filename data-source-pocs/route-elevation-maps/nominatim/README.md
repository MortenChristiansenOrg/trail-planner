# Nominatim

Run:

```bash
python3 data-source-pocs/route-elevation-maps/nominatim/poc.py
```

Saved samples:

- `raw/sample-gjendesheim.json`
- `raw/sample-skjeggedal.json`
- `raw/sample-fort-william.json`
- `normalized/sample-place-search.json`

This POC uses the public `nominatim.openstreetmap.org` endpoint with a low-volume test and a one-request-per-second delay.
