# Local Shuttles And Boats POC

Manual-source evaluation for trail-specific shuttle, tourist bus, and boat segments.

This folder records source links and normalized manual transport segments where public pages allow human-readable access. No scraping integration is assumed.

- https://www.trolltunga.com/en/parking-and-transport/shuttle-busses
